# Drive-Upload

## Installation
1. Clone the repository.<br />
`git clone https://github.com/TheBikramLama/DriveUpload`
2. Make **init.sh** executable.<br />
`chmod +x init.sh`
3. Run **init.sh**<br />
`./init.sh`
